Бот разрабатывается для облегчения работы на \href{https://discord.gg/yycb4pSRu5}{\texttt{ сервере}} {\bfseries{Артёма}}\+: канал \href{https://www.youtube.com/@Laidfin}{\texttt{ Laidfin}}, предоставляя ряд полезных функций. \begin{DoxyEnumerate}
\item Прочти все мануалы из раздела \href{https://github.com/Laidfinland-Team/Laidinen-Bot\#\%D0\%B4\%D0\%BB\%D1\%8F-\%D1\%80\%D0\%B0\%D0\%B7\%D1\%80\%D0\%B0\%D0\%B1\%D0\%BE\%D1\%82\%D1\%87\%D0\%B8\%D0\%BA\%D0\%BE\%D0\%B2}{\texttt{ Для разработчиков}}

\item Получи добро на добавление нового функционала
\item Форкни себе репозиторий
\item Создайте ветку для новой функциональности (от dev ветки)
\item Добавь новую команду или исправь обнаруженные баги
\item Добавь doxygen строки для новых функций
\item Обнови README.\+md (если это необходимо)
\item Обнови AUTHORS.\+md, Добавь своё имя в раздел в который тебя направили после одобрения запроса
\item Создай pull request в dev ветку (пожалуйста, заполни шаблон pull request)
